Included Samples

The samples included with Intel® Visual Fortran are designed to illustrate specific compiler optimizations, features, tools, and programming concepts.

Each group of samples is provided within a ZIP archive file. Expand the ZIP file to a writable directory of your choice. All samples are provided as Microsoft Visual Studio solutions and also contain command-line build instructions in the sources or in a build.bat file. Some samples include a "read me" file containing important information on building and running the sample.

Some of these samples are meant to illustrate features during compilation only and do not result in executables. Microsoft Visual Studio 2005* solutions are provided.

Intel® Visual Fortran Programming Samples

These samples are provided as Microsoft Visual Studio .NET 2003* solutions (and can be used with Microsoft Visual Studio 2005* or 2008), along with a build file that builds the Fortran (and C where provided) files from the command line. Some of these samples have a readme.txt file included.
