WHIZZY example for Intel Visual Fortran

This program demonstrates a variety of dialog box controls using the IFLOGM module.

If you have trouble building this project, make sure that the following folders,
or their equivalents, are listed under Tools..Options..Intel Fortran..Project
Directories..Includes:

  C:\Program Files\Microsoft Visual Studio .NET 2003\Vc7\atlmfc\include
  C:\Program Files\Microsoft Visual Studio .NET 2003\Vc7\PlatformSDK\Include
